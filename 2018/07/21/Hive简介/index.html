Hive简介

Hive是一个基于Hadoop的开源数据仓库（注意不是数据库），用于存储和处理海量结构化数据，它是Facebook 2008年8月开源的一个数据仓库框架。

  • 使用HQL（类似SQL语言）作为查询接口
  • 使用HDFS存储数据
  • 使用Mapreduce计算
  • Hive只能处理表结构的数据
    其本质是将HQL语言转化为Mapreduce程序。

常用命令

1.查看所选择表格的详细信息# desc formatted TableName;
2.插入本地数据至指定表格# load data local inpath '/Paths' into table DatabaseName.TableName;
3.展示所有Hive自带函数#show functions;；显示某个函数的帮助信息#desc function extended FunctionName;
4.执行外部sql命令#bin/hive -f sql.txt；保存结果至外部文本##bin/hive -f sql.txt >　save.txt
5.在Hive中也可使用dfs命令，如# dfs -ls /；也可执行Linux命令，如#!ls /modules
6.启动Hive时启用某种属性（仅对该次启动生效）,如#bin/hive --hiveconf hive.cli.print.current.db=true(显示当前数据库名)；也可在进入Hive后设置，如#set hive.cli.print.current.db=false;
7.定义变量：可在启动Hive时定义，如：#bin/hive --hivevar myname=haze;也可在启动后定义，如：#set key=val
